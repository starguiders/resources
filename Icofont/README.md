# IcoFont
An icon font made by [IcoFont](https://icofont.com/), modified by me to use in my sites.

# Usage

* add the stylesheet to your html page before the end of head with: `<link rel="stylesheet" href="https://starguiders.github.io/resources/Icofont/Icofont.css"/>`
* add an icon: `<i class="icofont-anchor"></i>`, substituting after icofont- with the icon name.

Check out the demo over here: **[CLICK ME](https://starguiders.github.io/resources/Icofont/demo.html).**
