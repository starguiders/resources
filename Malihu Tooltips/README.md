# Malihu Tooltips
A jQuery plugin made by [Malihu](https://manos.malihu.gr/style-my-tooltips-jquery-plugin/), and updated by [Malihu](https://breezescodes.tumblr.com/post/634232920475893760/updated-malihu-tooltip-code). Hosted by me because it's easier to keep track of.

# Usage

* add the script to your html page before </head> with: `<script src="https://starguiders.github.io/personal/Malihu%20Tooltips/style-my-tootltips.js"></script> `
* add this to your main js code, NOT with a <script></script>: `(function($){$(window).load(function(){ setTimeout(function() { $("[title]").style_my_tooltips({tip_follows_cursor:true,tip_delay_time:0,tip_fade_speed:300 }); }, 400); });})(jQuery);`
* copy and paste the css [from over here](https://starguiders.github.io/personal/Malihu%20Tooltips/style-my-tootltips.css) wherever your css goes and modify to your heart's content.
